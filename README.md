# nh-guide

A guide to New Hampshire wildlife written in LaTeX with data scraped from
http://amc-nh.org/resources/guides/

## To use

### Web scraper

* `python scraper.py`

### LaTeX guide builder

* `lualatex guide.tex`
* `./cleanup.sh`
